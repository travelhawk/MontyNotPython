# Monty Not Python

**Monty Not Python** was developed during an university course. Monty is a robot which has a camera to observe and document your creative work. The camera of the robot follows your movement and take screenshots during low-action movement pauses. The aim is to have a documentation of the process of your work. The hardware of the robot consists of a RaspberryPi, a camera and ultrasound sensors. It is possible to remote control the robot using an Android application.

The folder *MontyRaspberryPI* contains the application for the Raspberry Pi.
The folder *RemoteControl* contains th Android application to remote control the robot.
